Как устанавливать из GitHub Releases

Android (APK)

1) Открой ссылку .apk → скачай файл.
2) Разреши установку из неизвестных источников (если спросит).
3) Открой APK и следуй инструкциям.

Windows (EXE/MSI)

1) Скачай файл .exe / .msi.
2) Дважды кликни → «Далее» → «Установить».
3) Если SmartScreen спрашивает — жми «Подробнее» → «Выполнить в любом случае» (для новых подписанных билдов он не появляется).

Нужны альтернативы (ZIP/Portable или Linux AppImage)? Могу добавить отдельные кнопки.
